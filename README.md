# AI & Machine Learning Journey

> *"The expert in anything was once a beginner who refused to quit."*

Welcome! 

This repository documents my journey from a Computer Science student to becoming an **AI/ML Engineer**.

This is not just a collection of notes or tutorials. It is a structured learning roadmap where I study concepts, build projects, solve exercises, and document my progress.

The goal is simple:

> **Learn deeply. Build consistently. Become employable.**

---

#  Philosophy

This repository follows a few simple principles.

* Learn one topic at a time.
* Understand concepts instead of memorizing syntax.
* Write code after every lesson.
* Build projects frequently.
* Track progress honestly.
* Focus on consistency rather than speed.

The aim is to build real engineering skills, not just complete online courses.

#  How to Use Each Learning Folder

Every learning folder follows the same structure.

### 1. Read the Roadmap

Start with the roadmap file.

Example:

```
numpy/numpy_roadmap.md
```

The roadmap contains:

* learning order
* topic priorities
* required depth (Skim / Working Knowledge / Deep Understanding)
* milestone projects
* mastery checkpoints

It acts as the tracker for the entire topic.

---

### 2. Read the Guide

Each folder also contains:

```
HOW_TO_USE_ROADMAP.md
```

This explains exactly how to study that subject.

It includes:

* how to approach each topic
* how much time to spend
* when to move on
* when to build projects
* how to judge mastery
* recommended practice strategy

Think of it as the instruction manual for the roadmap.

---

### 3. Learn → Practice → Build

For every topic:

1. Learn the concept.
2. Solve practice exercises.
3. Answer conceptual questions.
4. Build a small project.
5. Update the roadmap.

Only then move to the next topic.

---

#  Learning Method

For every concept I study, I follow this cycle:

```
Learn
   ↓
Understand
   ↓
Implement
   ↓
Practice
   ↓
Build
   ↓
Review
```

Watching videos alone does not count as learning.

If I cannot build something with a concept, I have not learned it yet.
